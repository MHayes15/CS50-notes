## Week 0 notes

#### Binary

- Computer's language: Binary (0's and 1's)
- 0's and 1's are known as bits
- In programming, count from 0
- Binary stored in transistors (switches)

##### Normal
100s place | 10s place | 1s place  
    1           2           3       **= 123**

##### Binary
4s place | 2s place | 1s place
    0          0          1         **= 1**
    0          1          0         **= 2**
    0          1          1         **= 3**
    1          0          0         **= 4**
    1          0          1         **= 5**
